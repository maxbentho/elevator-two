# Max Bentho – Intro

Minimal Astro build that introduces Max Bentho’s background, selected clients, and contact links.

## Getting started

```bash
npm install
npm run dev
```

## Commands

| Command         | Action                                   |
| --------------- | ---------------------------------------- |
| `npm run dev`   | Starts the local dev server              |
| `npm run build` | Builds the production bundle into `dist` |
| `npm run preview` | Serves the production build locally   |

## Tech stack

- [Astro](https://astro.build/)
- Vanilla CSS for layout and typography

## Deployment

Run `npm run build` and deploy the `dist` folder to your static host of choice.
